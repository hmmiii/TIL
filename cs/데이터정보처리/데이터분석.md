## 지식정보화사회

정보화사회, 이를 넘어서 현재는 지식정보화 사회라고 한다.

지식정보화사회에서는 컴퓨터를 다루는 능력을 넘어서, 데이터 분석 능력을 필요로 한다.

## 데이터 분석 절차

1. 문제 정의
2. 조사, 실험 계획
3. 데이터 수집
4. 데이터 정리, 분석
5. 분석 결과 평가

## 데이터 분석 소프트웨어

1. 엑셀
2. SPSS, SAS
    - SPSS는 GUI환경으로 스프레드시트를 제공해서 사용이 편리하다.
    - SAS는 언어 식으로 되어 있고, 통계 분석 언어이다.
3. R
    - 무료 오픈소스
    - 강력한 통계분석언어이다.
    - R Studio 툴을 사용한다.
